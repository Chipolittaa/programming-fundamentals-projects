# Bitwise and Logical Operations

## Описание
Программа на языке C для ОС Linux, которая **разворачивает порядок битов** в 64-битном числе.

Например:
```
Вход:  00000000 00000000 00000000 00000000 00000000 00000000 00000000 00001101  (число 13)
Выход: 10110000 00000000 00000000 00000000 00000000 00000000 00000000 00000000
```

### Что делает программа
1. Принимает число в десятичной системе счисления (или генерирует случайное)
2. Выводит его 64-битное двоичное представление
3. Разворачивает все 64 бита в обратном порядке
4. Выводит результат в двоичном виде

### Ключевые битовые операции
| Операция | Где используется | Зачем |
|----------|-----------------|-------|
| `num >> i` | `print_binary` | Сдвиг вправо — получить бит на позиции `i` |
| `& 1` | `print_binary` | Маскирование — выделить только младший бит |
| `reversed << 1` | `reverse_bits` | Сдвиг влево — освободить место для следующего бита |
| `\| (num & 1)` | `reverse_bits` | OR — записать младший бит исходного числа в результат |
| `num >>= 1` | `reverse_bits` | Сдвиг — перейти к следующему биту |

## Требования
- ОС: Linux
- Компилятор: `gcc`
- Утилита: `make`

---

## Сборка и запуск
### Сборка
```bash
make
```

### Запуск с числом
```bash
./prg1pmsN3146 <число>
```

### Запуск без аргумента (случайное число)
```bash
./prg1pmsN3146
```

### Очистка скомпилированных файлов
```bash
make clean
```

---

## Примеры
```bash
$ ./prg1pmsN3146 1
Исходное число в двоичном формате:
00000000 00000000 00000000 00000000 00000000 00000000 00000000 00000001 
Число после разворота битов в двоичном формате:
10000000 00000000 00000000 00000000 00000000 00000000 00000000 00000000 

$ ./prg1pmsN3146 255
Исходное число в двоичном формате:
00000000 00000000 00000000 00000000 00000000 00000000 00000000 11111111 
Число после разворота битов в двоичном формате:
11111111 00000000 00000000 00000000 00000000 00000000 00000000 00000000 
```

### Обработка ошибок
```bash
$ ./prg1pmsN3146 abc
Ошибка: 'abc' не является числом.
```
