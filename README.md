# Programming Fundamentals Projects
A collection of C and Python projects focused on programming fundamentals, memory management, data processing and software design.

---

## About
This repository contains practical programming projects developed to strengthen core software engineering skills. The projects cover low-level programming in C, object-oriented programming in Python, working with files, pointers, dynamic memory, iterators, generators and basic algorithmic thinking.

---

## Main Topics
- C programming
- Python programming
- Bitwise and logical operations
- Pointers and dynamic memory
- Strings and input/output
- File processing
- Pointer-based data structures
- Iterators and generators
- Object-oriented programming
- Makefile-based builds
